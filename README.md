# 🤖 AI Agency — Стартап-агенты для Claude

Набор промптов и фреймворков для запуска AI-агентной команды внутри Claude Projects. Каждый агент выполняет роль конкретного C-level руководителя и работает по строгим профессиональным принципам.

## Зачем это нужно

Вместо того чтобы каждый раз объяснять контекст с нуля, этот репозиторий даёт Claude готовые личности, фреймворки и инструменты для системного ведения стартапа.

## 🧑‍💼 Агенты

### CBO — Chief Brand Officer
Элитный бренд-стратег и творческий директор. Работает через концепции Golden Circle, Brand Archetypes (по Юнгу), Visual Semiotics. Отказывается начинать работу над визуалом без ответа на вопрос "Зачем мы существуем?". Борется с клише и заезженными формулировками.

**Артефакт в конце каждого ответа:** Бренд-радар (статус упаковки).

### COO — Chief Operating Officer
Жёсткий операционный директор. Применяет MECE-декомпозицию, теорию ограничений, Iron Triangle. Не принимает статус "почти готово". Переводит мечты в регламенты, таблицы и дедлайны.

**Артефакт в конце каждого ответа:** Статус операционки (фокус недели, узкое горлышко, ближайший дедлайн).

## 🚀 Как использовать

1. Создай **Claude Project**
2. Загрузи нужные файлы в раздел Project Knowledge:
   - Для брендинга: `brand_skills/00_brand_identity.md` + нужные фреймворки
   - Для операционки: `coo_skills/00_coo_identity.md` + нужные фреймворки
3. Начни диалог — агент автоматически принимает нужную роль
