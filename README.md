# 🤖 AI Agency — Стартап-агенты для Claude

Набор промптов и фреймворков для запуска AI-агентной команды внутри Claude Projects. Каждый агент выполняет роль конкретного C-level руководителя и работает по строгим профессиональным принципам.

## Зачем это нужно

Вместо того чтобы каждый раз объяснять контекст с нуля, этот репозиторий даёт Claude готовые личности, фреймворки и инструменты для системного ведения стартапа.

---

## 🗂 Структура

| Папка | Содержимое |
|---|---|
| `brand_skills/` | Промпты для CBO-агента (личность, нейминг, фундамент бренда) |
| `brand_frameworks/` | Шаблоны: платформа бренда, дизайн-бриф |
| `coo_skills/` | Промпты для COO-агента (личность, планирование) |
| `coo_frameworks/` | Шаблоны: Gantt, OKR-трекер, спецификации найма |

---

## 🧑‍💼 Агенты

### CBO — Chief Brand Officer
Элитный бренд-стратег и творческий директор. Работает через Golden Circle, Brand Archetypes (по Юнгу), Visual Semiotics. Отказывается начинать работу над визуалом без ответа на вопрос "Зачем мы существуем?". В конце каждого ответа выводит **Бренд-радар** — статус упаковки.

### COO — Chief Operating Officer
Жёсткий операционный директор. Применяет MECE-декомпозицию, теорию ограничений, Iron Triangle. Не принимает статус "почти готово". В конце каждого ответа выводит **Статус операционки** — фокус, узкое горлышко, дедлайн.

---

## 🚀 Как использовать

1. Создай **Claude Project**
2. Загрузи в Project Knowledge нужные файлы:
   - Брендинг → `brand_skills/00_brand_identity.md` + фреймворки из `brand_frameworks/`
   - Операционка → `coo_skills/00_coo_identity.md` + фреймворки из `coo_frameworks/`
3. Начни диалог — агент автоматически принимает роль
